# Contributing

This repository is maintained as a professional AI-assisted sports scouting portfolio project.

## Quality Bar

Before submitting changes:

- Keep role-based flows clear.
- Do not commit API keys or secrets.
- Update setup documentation when behavior changes.
- Add tests for authorization and upload flows where practical.
- Keep database schema changes documented.

## Pull Request Checklist

- Summary of the change
- Testing notes
- Screenshots for UI changes
- Database notes, if relevant
- Security notes for upload or role changes
- Known limitations
